Open-source Codex plugin

Structured workflows for Codex teams

Codex DeepWork packages skills, hooks, and MCP configuration so Codex can run repeatable, quality-gated project workflows in any repository.

Install How it works

Why

More durable than a giant prompt

A one-off prompt can work once. A DeepWork-style job preserves steps, quality gates, repo conventions, and lessons learned so the same workflow can run again with less drift.

Run jobs

Start, continue, and complete workflows through the DeepWork MCP runtime.

Create jobs

Turn recurring work into `.deepwork/jobs` with Codex-native instructions.

Learn from runs

Capture corrections and improve workflows after real project use.

Install

Add it once, use it in any repo

The plugin lives outside your project. The jobs and generated outputs stay inside whichever repository Codex is working in. 

codex plugin marketplace add /path/to/codex-deepwork
codex plugin add codex-deepwork@codex-deepwork
